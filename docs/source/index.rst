Welcome to the Facility Reconciliation Tool's documentation!
============================================================

The Facility Reconciliation Tool is a part of the Global Open Facility Registry. It's and open source and open standards-based tool to help users match lists of facilities from different data sources. The tools supports uploading data from CSV, FHIR servers and DHIS2 instances.

If you are new to the tools, please consider trying the quick starts!
